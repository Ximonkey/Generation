Bootstrap-Introducción

Clases principales y de mayor uso en Bootstrap

A continuación se muestran algunas clases importantes para utilizar en bootstrap:

class="container"

Ocupa las 12 columnas del grid que proporciona CSS, mantiene un margen a los costados.

class="container-fluid"

Ocupa el tamaño completo del viewport.

class="row"

El elemento contenedor se comporta como una fila dentro del Grid.

class="col-n/xx"

Establece el numero de columnas para el elemento. Se puede delimitar con numero o nomenclatura definida: sm, md, lg, xl, xxl.
